git init: Inicializa un nuevo repositorio Git en el directorio actual.
git clone -URL-: Clona un repositorio Git existente desde una URL.
git add -archivo: Agrega un archivo al área de preparación (staging).
git add . o git add --all: Agrega todos los archivos nuevos y modificados al área de preparación.
git commit -m "Mensaje del commit": Crea un nuevo commit con los archivos en el área de preparación y un mensaje descriptivo.
git status: Muestra el estado actual de tu repositorio, incluyendo archivos sin seguimiento, modificados y en el área de preparación.
git log: Muestra un registro de los commits en el repositorio.
git branch: Lista las ramas en el repositorio y muestra la rama actual.
git branch -nombre-rama-: Crea una nueva rama.
git checkout -nombre-rama-: Cambia a una rama diferente.

git merge -nombre-rama-: Fusiona una rama con la rama actual.
git pull: Descarga cambios desde un repositorio remoto y los fusiona en tu rama actual.
git push: Sube tus cambios locales a un repositorio remoto.
git remote -v: Muestra la lista de repositorios remotos configurados.
git fetch: Descarga cambios desde un repositorio remoto sin fusionarlos en tu rama actual.
git diff: Muestra las diferencias entre el área de preparación y el directorio de trabajo.
git reset -archivo-: Quita un archivo del área de preparación.
git reset --hard -commit-: Restablece el repositorio al estado de un commit específico.
git tag -nombre-: Crea una etiqueta para un commit específico.
git rm -archivo-: Elimina un archivo del control de versiones y del sistema de archivos.
