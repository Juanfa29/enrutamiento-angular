Guia Para Instalar Angular

1. Node.js: Asegúrate de que Node.js esté instalado en tu sistema. Puedes descargarlo desde

el enlace del sitio web oficial de Node.js.

2. NPM (Node Package Manager): NPM generalmente se instala junto con Node.js. Verifica si NPM está instalado ejecutando el siguiente comando en tu terminal:
"npm -v"
Debería mostrarte la versión de NPM instalada.

Pasos para instalar Angular:

1. Abre tu terminal o línea de comandos. Asegúrate de tener acceso a una conexión a Internet
ya que Angular se instala a través de NPM.
2. El siguiente comando para instalar Angular CLI (Command Line Interface):
npm install -g @angular/cli
Este comando instalará Angular CLI globalmente en tu sistema, lo que te permitirá crear y administrar proyectos de Angular.
3. Una vez que la instalación se haya completado, verifica que Angular CLI se haya instalado correctamente.
Ejecuta el siguiente comando:
ng --version
Debería mostrarte la versión de Angular CLI instalada y otra información relevante.
4. Ahora estás listo para crear un nuevo proyecto de Angular.

Como crear un nuevo componente en Angular

 1. Para crear un componente nuevo usando el Angular CLI, ejecuta el siguiente comando en la raíz de tu proyecto:
generate component nombre-del-componente
Por ejemplo, para crear un componente llamado mi-componente, ejecutarías el siguiente comando:
ng generate component mi-componente
Este comando creará los siguientes archivos:
mi-componente.component.ts: El archivo de TypeScript que contiene la clase del componente.
mi-componente.component.html: El archivo HTML que contiene la plantilla del componente.
mi-componente.component.css: El archivo CSS que contiene el estilo del componente.

Comandos básicos que se utilizan en Angular
Los comandos básicos que se utilizan en Angular son los siguientes:

ng new - Crea un nuevo proyecto de Angular.
ng serve - Inicia el servidor de desarrollo y sirve la aplicación.
ng build - Compila la aplicación para producción.
ng test - Ejecuta las pruebas unitarias de la aplicación.
ng e2e - Ejecuta las pruebas de extremo a extremo de la aplicación.

¿Que es Angular?
Angular es un framework de aplicaciones web de código abierto desarrollado por Google. Está escrito en TypeScript y permite crear aplicaciones web de una sola página (SPA)
que son rápidas, escalables y fáciles de mantener.Utiliza una arquitectura basada en componentes para organizar el código de la aplicación. Los componentes son
bloques de construcción reutilizables que pueden usarse para crear diferentes partes de la aplicación.Es una buena opción para aplicaciones web que requieren una buena
experiencia de usuario, escalabilidad y facilidad de mantenimiento.
